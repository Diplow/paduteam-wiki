# Allégria

## Position dans le [[Graphique]]
Organisation en haut à droite — école privée de code et d'IA.

## Dynamique
"Première école code et IA destinée aux particuliers", lancée par [[Francis Lelon]] (cofondateur de Sarenza). Pour la PaduTeam, Allégria incarne le projet idéologique de [[Connaissance vs compétences]] : des compétences directement exploitables en entreprise (code, IA) à la place de connaissances générales. C'est l'école élitiste pénurique sous forme de startup.
