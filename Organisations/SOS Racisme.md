# SOS Racisme

## Profil
Association antiraciste française fondée en 1984 ("Touche pas à mon pote"). Référence historique dans l'analyse PaduTeam de l'[[Antiracisme politique vs antiracisme moral|antiracisme abstrait/universaliste]].

## Position dans le [[Graphique]]
Représente un antiracisme des classes moyennes-hautes, sans ancrage dans les classes populaires racisées. À l'époque de sa fondation, il n'existait pas de clivage race/gauche dans les classes ouvrières — l'extrême droite n'était pas encore hégémonique dans les classes populaires.
