# CHARLIE HEBDO EST-IL RÉACTIONNAIRE ? (GRAPHOMARXISME avec Chris de Positions Revue)

## Résumé

Format Graphomarxisme avec [[Chris]] de [[Positions Revue]] : un sondage Ifop sur le droit à la caricature — publié à l'occasion des 10 ans de l'attentat — est repondéré selon les PCS et projeté sur le [[Graphique]]. Résultat sans appel : l'esprit Charlie est un **clivage de classe**, avec les partisans de la caricature sans limite concentrés dans les classes supérieures. La vidéo démontre simultanément la méthode de lecture des clivages (parallèle vs perpendiculaire à la diagonale) et tranche une question politique récurrente : Charlie Hebdo n'est pas une position universelle de gauche mais l'idéologie d'une bourgeoisie progressiste urbaine qui mobilise la liberté d'expression pour dominer ceux qui subissent les caricatures.

## Thèses et analyses clés

1. **Charlie = clivage de classe, pas un clivage universel** — La diagonale oui/non (pour/contre la caricature) est **parallèle** à la diagonale de classe du [[Graphique]], non perpendiculaire. Cela signifie que les positions sur Charlie ne transcendent pas les classes sociales : elles *expriment* un rapport de classe. Les classes supérieures défendent la caricature illimitée ; les classes populaires, qui subissent ces caricatures, s'y opposent davantage[^1].

2. **Méthode graphique pour identifier un clivage** — La PaduTeam expose le critère diagnostique central : si la ligne entre les pôles oui/non est **parallèle** à la diagonale bas-gauche → haut-droite, le clivage est de classe. Si elle est **perpendiculaire** (haut-gauche → bas-droite), c'est un clivage transclasse (générationnel, de mode de production, de genre). Application comparée : le cannabis (perpendiculaire = [[Clivage generationnel]]) vs Charlie (parallèle = clivage de classe)[^2].

3. **Le décalage générationnel dans le clivage Charlie** — Si le clivage est principalement de classe, il est aussi légèrement excentré vers les vieux : le "non" (pour la caricature) est tiré vers les classes supérieures *retraitées*, le "oui" (contre) étant plus central, tiré vers les retraités de classes inférieures. Résultat : les deux diagonales sont parallèles mais pas sur la même ligne — Charlie intéresse davantage les vieux[^3].

4. **Deux racismes structurellement différents sur le Graphique** — Le racisme de pénurie (bas-gauche) est un racisme concurrentiel sur les ressources : les classes populaires peuvent refuser les caricatures de Mahomet parce qu'elles ont un rapport plus collectif aux choses, pas parce qu'elles sont islamistes. À l'inverse, le racisme civilisationnel (haut-droite : [[Eric Zemmour|Zemmour]], [[Printemps Republicain]]) impose ses valeurs universelles aux autres civilisations. Charlie appartient à cette seconde matrice[^4].

5. **L'universalisme abstrait bourgeois comme instrument de domination** — L'esprit Charlie défend la liberté d'expression "qui n'engage que lui" — une liberté individualiste abstraite qui fait abstraction des conséquences sur les classes racisées et populaires qui *subissent* ces caricatures. C'est l'"antiracisme universaliste abstrait" des classes supérieures blanches, qui se pose en arbitre de ce qui peut être dit contre les croyances des autres[^5].

6. **Fabien Roussel dans l'espace Charlie** — Sur le Graphique, [[Fabien Roussel]] se retrouve dans le même espace que [[Franc-Tireur]] et [[Raphael Enthoven]] — du côté "non" (pour la caricature illimitée). La PaduTeam analyse : Roussel est "plus hégélien que léniniste" sur cette question, préférant défendre la laïcité abstraite dans son absolu historique plutôt que penser concrètement que le débat Charlie est "une forme de violence contre les classes les plus opprimées."[^6]

7. **LFI dans l'espace "oui" — un non-sujet stratégique** — La France Insoumise est positionnée dans l'espace "oui" (contre la caricature illimitée, à 53% selon le sondage). Mais surtout, c'est un **non-sujet** pour l'espace électoral LFI et Le Pen : avoir un programme complet sur la laïcité ne sert à rien pour capter les classes populaires, qui ne se polarisent pas sur ce thème. Marine Le Pen elle-même parle peu de laïcité — bien moins que Zemmour ou Gabriel Attal[^7].

8. **Franc-Tireur = autant islamophobe que Zemmour mais se croit encore de gauche** — La PaduTeam formule le diagnostic : Franc-Tireur et ses figures (Enthoven...) sont "autant islamophobes que Zemmour mais qui se croient encore de la gauche des Lumières comme Macron." Cette catégorie se pense progressiste mais occupe exactement le même espace de classe réactionnaire[^8].
