# Possibles — Les horizons que la PaduTeam défend

Cette couche regroupe les **visions du monde positives** que la PaduTeam articule. Un possible n'est pas un combat (Enjeu) — un combat se mène *contre* un adversaire ; un possible se déploie *vers* un horizon. Le combat est défensif ou offensif ; le possible est constructif.

Un possible se reconnaît à ce qu'il :
- Énonce un horizon désirable, pas seulement une critique
- Articule un mécanisme matériel (comment on y va, pas juste que c'est bien)
- Affronte une vision concurrente (le possible adverse)

Exemples : le choc d'abondance contre la vision pénurique, l'universalisme matériel contre la hiérarchie raciale et nationale, le socialisme autogestionnaire (yougoslave), la désagrégation progressiste de l'empire américain comme horizon anti-impérialiste, le devenir-majoritaire de la gauche radicale.
