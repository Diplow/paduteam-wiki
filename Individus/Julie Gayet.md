# Julie Gayet

## Profil synthétique
Actrice française, compagne de [[Francois Hollande]]. Présente dans le reportage Paris Match sur leur vie quotidienne à Tulle.

## Analyse PaduTeam
Dans le reportage Paris Match, Julie Gayet est mise en scène en mode "trad wife" forcée : robe longueur midi, chignon approximatif, talons raisonnables, manipulation de fleurs dans la cuisine. Padu : "On lui a dit Julie, tu vas te foutre dans la cuisine." Le décor est entièrement fabriqué — jardinier qui dépose les fleurs, petites touches bourgeoises disposées pour paraître spontanées.

Sa présence dans le reportage est analysée comme un outil de normalisation de Hollande : faire passer un ex-président que "le pays déteste" pour "un homme simple" via la mise en scène d'un couple ordinaire.
