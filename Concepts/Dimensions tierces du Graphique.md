# Dimensions tierces du Graphique

## Définition
Le [[Graphique]] est en 2D (axe exploitation × domination), mais genre et race y sont encodés implicitement : chaque rond (PCS) a une composition interne genrée et racisée. Les "dimensions tierces" désignent ces distributions à l'intérieur des ronds.

## Mécanisme
Chaque PCS n'est pas un bloc homogène. À l'intérieur d'un même rond :
- certaines PCS sont très masculines (ex: anciens ouvriers, retraités ouvriers)
- d'autres sont plus féminines (ex: anciens employés)
- la proportion de personnes racisées varie fortement selon la position dans le graphique

**La règle empirique** : plus on descend vers les classes populaires (bas-gauche), plus la proportion de racisés est élevée — car l'immigration des années 1960-70 a produit des travailleurs dans les classes ouvrières et employées, pas dans les classes dominantes.

## Ce que ça révèle
- Quand deux ronds se touchent (ex: Mélenchon + abstention + Le Pen en 2017-2022), le point de jonction est un **clivage de race ou de genre** — quelque chose qui transcende les axes classe.
- Un projet féministe sera **plus hégémonique** dans les PCS à 50/50 que dans les PCS très masculines (antagonisme dissymétrique).
- L'antiracisme est **incontournable** dans les classes populaires, inutile dans les classes dominantes blanches où le clivage racial n'existe pas.
