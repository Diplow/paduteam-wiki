# Pipeline class first

## Définition

Le **pipeline class first** désigne la trajectoire de déradicalisation de l'extrême droite vers la gauche radicale par l'entrée de la **conscience de classe**. C'est l'inverse symétrique du pipeline masculiniste/fasciste : quand le pipeline réactionnaire capture les jeunes hommes de milieu populaire avec des explications identitaires (les immigrés, les féministes, les "wokes"), le pipeline class first leur offre une explication matérialiste (les patrons, le capital, les rapports de production).

## Comment ça fonctionne

Trajectory type :[^1]

1. Jeune homme de milieu populaire, souvent blanc, souvent désocialisé
2. Capturé par le pipeline réactionnaire : extrême droite internet, masculinisme, identitarisme
3. Rencontre avec un cadre d'analyse matérialiste (classe sociale, rapports de production, antagonisme de classe)
4. Réorientation progressive : les ennemis réels ne sont pas les immigrés ou les féministes mais les exploiteurs
5. Transition vers la gauche radicale

## Conditions de possibilité

Le pipeline class first ne fonctionne pas seul — il nécessite :[^2]

- Des organisations capables de faire ce travail de formation politique (La Brèche, syndicats, partis)
- Un tissu militant dense qui puisse accueillir les personnes en transition
- Une division du travail entre organisations : certaines font la déradicalisation, d'autres font le travail politique, d'autres font le soutien (exemple : Guirot qui oriente [[Infame]] vers l'[[Organisation Solidarite Trans|OST]])

## Limites

La PaduTeam ne présente pas le pipeline class first comme une solution universelle :[^3]

- Il ne fonctionne que si les organisations existent pour le faire
- Il ne fonctionne que sur certaines personnes (celles dont l'engagement à l'extrême droite est encore superficiel ou en cours de formation)
- L'humour peut parfois servir de transition (cas d'[[Infame]]) mais aussi de couverture à des positions réactionnaires persistantes
