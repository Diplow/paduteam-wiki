# Frérisme

## Définition

Terme désignant l'infiltration supposée de la société française par les Frères Musulmans. Dans le discours médiatique mainstream (CNews, rapport parlementaire, [[Florence Bergeaud-Blackler]]), le "frérisme" désigne une stratégie d'entrisme : des individus ou organisations liés aux Frères Musulmans s'infiltreraient dans les institutions, la culture et le tissu social français pour les islamiser progressivement.

Pour la PaduTeam, le "frérisme" comme concept opératoire dans le débat public est une **fiction complotiste** fonctionnellement identique à l'antisémitisme du XIXe siècle. Ce n'est pas une analyse — c'est un outil idéologique de stigmatisation des musulmans.

## Les Frères Musulmans : ce qu'ils sont réellement

Les Frères Musulmans sont apparus en Égypte au XXe siècle, dans un contexte de fracture entre élites laïques enrichies et classes populaires encore très religieuses. C'est d'abord un **mouvement social** qui porte une question de justice sociale dans un cadre islamique. La PaduTeam cite également le FIS algérien comme mouvement analogue : né après l'indépendance pour contrecarrer les échecs sociaux du FLN.

Ces mouvements n'ont jamais pris le pouvoir dans des pays où ils étaient massivement implantés et où ils représentaient des millions de personnes (Égypte, Algérie). Prétendre qu'ils vont le faire en France, où ils représentent 3-5 % des mosquées (soit environ 0,1 % des musulmans français), relève du délire.

## La logique complotiste du frérisme

Le raisonnement frériste est structurellement fermé — il partage la même logique que toutes les théories complotistes : l'absence de preuve est elle-même une preuve. [[Florence Bergeaud-Blackler]] dans son livre sur les Frères Musulmans affirme des choses qu'elle ne peut pas prouver "parce que les frères musulmans ont effacé les preuves". C'est le raisonnement type : "je le prouve pas parce que toute façon ils ont effacé les preuves."

Conséquence : la catégorie "frériste" est indéfiniment extensible. Tout ce qui n'est pas islamophobe devient suspect de frérisme. [[George Soros]] serait "de mèche avec les frères musulmans" parce qu'il a financé un projet demandant qu'on laisse les musulmans exprimer leurs revendications. Un rabin sioniste de gauche ([[Emile Ackerman]]) serait "influencé par les frères musulmans" parce qu'il prône une "laïcité adjectivée".

La PaduTeam formule la conclusion logique : "soit tu es islamophobe, soit tu es frériste" — il n'y a pas de troisième position.

## Frérisme = nouvel antisémitisme

La PaduTeam établit une analogie systématique avec l'antisémitisme du XIXe siècle et de l'Affaire Dreyfus :

- Même accusation de "nation dans la nation" (les Juifs = agents de l'Allemagne / les musulmans = agents des Frères Musulmans)
- Même logique de la 5e colonne : "ces gens veulent détruire la France de l'intérieur"
- Même extensibilité : quiconque défend les accusés devient suspect
- Même impossibilité à se défendre : la dénégation est elle-même une preuve d'infiltration

La PaduTeam : "On vit vraiment l'antisémitisme nouveau, celui qu'on a vécu au 19e siècle." Elle espère qu'une "affaire Dreyfus pour un compatriote musulman" permette de sortir de ce cycle, comme l'affaire Dreyfus a finalement unifié la gauche contre l'antisémitisme.

## Les chiffres réels (rapport parlementaire français)

Le rapport parlementaire sur les Frères Musulmans en France révèle :
- 139 mosquées "problématiques" sur 2 800 (≈ 5 % des mosquées)
- 127 associations "fréristes" sur 325 000 associations (0,004 %)

Ces chiffres représentent une situation **normale** pour n'importe quel courant minoritaire dans une communauté religieuse. La PaduTeam compare : les intégristes catholiques (sédévacantistes, traditionalistes) représentent probablement des proportions similaires parmi les chrétiens — sans que personne n'en fasse un "danger pour la République".

## Fonctions idéologiques du frérisme

Le recours au "frérisme" comme catégorie politique remplit plusieurs fonctions pour la bourgeoisie en crise :

1. **Diviser le prolétariat** : séparer la fraction musulmane des classes populaires du reste du salariat, empêchant toute coalition révolutionnaire (voir [[Islamophobie]])
2. **Faire diversion** : déplacer le débat des questions sociales (inégalités, salaires, services publics) vers des questions identitaires qui n'intéressent pas la majorité
3. **Justifier la répression** : créer un ennemi intérieur permanent qui légitime la surveillance, les lois d'exception, les discriminations institutionnalisées

## Lien avec la laïcité culturaliste

Le frérisme est le versant "complotiste" de la [[Laicite culturaliste]] : si la laïcité culturaliste dit "la pratique islamique visible est un problème culturel", le frérisme dit "la pratique islamique visible est un projet politique organisé de conquête du pouvoir". Les deux visent le même effet : délégitimer les pratiques religieuses musulmanes dans l'espace public.
